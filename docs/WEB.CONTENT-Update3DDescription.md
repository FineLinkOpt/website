# 任務紀錄: 更新 3D 輪廓量測設備產品說明

## 任務描述
根據使用者需求，更新「3D 輪廓量測設備」產品頁面 (3d-profiler.html) 的核心說明文案。

## 修改內容
- **檔案**: `3d-profiler.html`
- **修改位置**: 
    - `<meta name="description">`
    - Hero 區域的 `<p class="hero-desc">`
    - 將應用領域區塊進一步改寫為與前三點相同的 `<li>` 特點項目格式，統整為「工業級決策視覺平台」。
    - 「直覺的 3D 視覺化分析」區塊的圖片說明 (`.gallery-item`)
- **新文案**: 
    > 專為高反射、透明、軟性材料與精密結構量測需求所打造，提供非接觸式高精度輪廓量測能力。（為避免與下方第四項特色介紹重複，已刪除「可應用於翹曲、厚度、平坦度、表面輪廓與精密結構檢測」這段冗長文字）
- **新增與修正內容**:
    > 1. 新增「工業級決策視覺平台」說明，內容精簡為：「整合 AI 偵測及 OCR 功能。可應用於：偏光板翹曲量測、軟性材料平坦度檢測、精密結構高度差量測、表面輪廓分析、高反射材料量測。」此內容已完全融入上方的設備特色清單中，成為第四個亮點項目。
    > 2. 將視覺化分析區塊中展示 3D 拼接圖的項目標題更正為「**大視野 3D 拼接分析**」，說明修改為「結合高精度拼接演算法，突破單鏡頭視野極限，完整還原大尺寸元件的 3D 全貌，精準評估整體的翹曲與平整度。」以完全符合圖片實際展示的技術內容。
    > 3. 修正「軟體核心功能與演算法」區塊中「多頻相位解碼 (Multi-Frequency FPP)」的演算法描述，移除錯誤的 FFT 相關敘述，更正為：「**結合專屬相移演算法與強效抗噪濾波，精準重建微米級 3D 高度圖**」。
    > 4. 將「大視野拼接技術 (Stitching)」的應用描述從單一的「適用大尺寸偏光板檢測」修改為具備高度泛用性的：「**支援多圖高精度拼接，突破單鏡頭視野限制，輕鬆應對大尺寸工件的嚴苛檢驗。**」
    > 5. 移除「軟體核心功能與演算法」區塊中所有標題後方的英文備註 (包含 Multi-Frequency FPP, HDR Fusion, Stitching, Template Editor 等)，讓整體中文排版更俐落純粹。
    > 6. 更新設備介紹區塊的「微米級精度掃描」描述，改為：「**運用 FPP 3D 結構光技術，高解析重建表面形貌，精準捕捉微細瑕疵。**」使語句更專業通順。
    > 7. 更新設備介紹區塊的第二個項目，將標題「專為極端應用打造」改為「**專為嚴苛檢測需求打造**」，說明修改為「**完美應對偏光板的翹曲變形與記憶體模組的複雜結構，實現高穩定度的精密量測。**」以提升工業級信任感。
